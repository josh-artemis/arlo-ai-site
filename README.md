# Arlo Health Platform Site

Arlo's marketing site for its **clinical action API for AI health platforms**.

One API to prescribe, order labs, and refer patients — licensed clinicians in all 50 states, HIPAA compliant. Built for AI health companies integrating clinical actions into their products.

## Contents

- `index.html` — Beta landing page
- `style.css` — Styles
- `script.js` — Client-side behavior
- `llms.txt` — LLM-readable product description
- `favicon.svg`, `logo.png`, `logo.svg` — Brand assets

## Local development

Open `index.html` in a browser or serve the folder with any static file server.

## Deployment & custom domain

The site deploys to GitHub Pages via the `Deploy Arlo to GitHub Pages` workflow. See [DOMAIN_SETUP.md](./DOMAIN_SETUP.md) for connecting **arlohealth.ai** (Namecheap).
